TEMA : website Donasi
	Proyek ini merupakan pengembangan Sistem Informasi Donasi ZISWAF dan Kotak Amal berbasis web yang dibuat selama kegiatan magang di Yayasan Nur Hidayah Surakarta. Sistem ini dirancang untuk mengatasi permasalahan pengelolaan donasi yang sebelumnya masih dilakukan secara manual dan terfragmentasi di berbagai cabang (LKSA).
	Tujuan utama dari proyek ini adalah untuk membangun sistem yang mampu memusatkan data donasi secara terintegrasi, meningkatkan efisiensi operasional, serta mendukung transparansi dan akuntabilitas dalam pengelolaan dana Zakat, Infak, Sedekah, Wakaf (ZISWAF), dan Kotak Amal.Sistem ini dikembangkan menggunakan teknologi PHP Native dan MySQL, serta menerapkan metode pengembangan Prototype untuk memastikan kebutuhan pengguna dapat terpenuhi secara optimal melalui proses evaluasi berulang.
	Hasil dari proyek ini adalah sebuah sistem yang mampu mengelola data donasi multi-cabang, melakukan pencatatan transaksi secara real-time, serta menyediakan laporan yang terstruktur dan mudah diakses oleh pihak yayasan. Melalui proyek ini, saya mendapatkan pengalaman dalam mengembangkan sistem informasi berbasis web secara end-to-end, mulai dari analisis kebutuhan, perancangan sistem, pengembangan aplikasi, hingga tahap pengujian dan implementasi langsung di lingkungan instansi.
